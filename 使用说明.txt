
🎙️使用方法

1. 程序本体就在 VRChatEasyTrans-Release 文件夹，不需要安装，可以直接把这个文件夹放在合适的位置
2. 进入 VRChatEasyTrans-Release 文件夹，找到 VRChatEasyTrans-AI.exe，双击运行即可。可以创建快捷方式到桌面方便以后运行双击运行
3. 填写你的配置，以及讯飞星火和 Deepseek 的 API 认证信息（下面会讲怎么获取）
4. 点击启动，这时配置会保存，下次启动无需再次配置

- 讯飞星火配置
1. 注册讯飞开放平台：https://www.xfyun.cn
2. 登录后点击右上角的“控制台”
3. 创建一个应用，进入应用
4. 左边栏点击“语音识别”模块下的“语音听写”
5. 右边 Websocket服务接口认证信息，可以看到你创建的应用的APPID，APISecret 和 APIKey，复制到 VRChatEasyTrans 中对应的讯飞星火部分。
6. 每天有500次的免费额度，也就是500句话，应该够用了吧。。不够用的自己买

- Deepseek 配置
1. 注册 Deepseek开放平台：https://platform.deepseek.com
2. 登陆后点击左边栏的 API kyes
3. 点击 创建API key。注意，创建后API key只会立刻显示一次，请立即复制保存！将 API key 复制到 VRChatEasyTrans 中对应的Deepseek部分。
4. 点击左边栏的 用量信息 ，查看账户余额
5. 没有余额的话点击左边栏 充值
6. 先充值 10 块钱就行，能用好久

⚠️⚠️⚠️ 注意 ⚠️⚠️⚠️
请勿将程序本体直接进行复制和转发，这会造成你的API密钥泄露，由此产生的任何损失我概不负责
