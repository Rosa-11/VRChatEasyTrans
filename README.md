# VRCEasyTrans
一个基于 C++ Qt 框架的实时翻译软件，轻量化实现实时的普通话/英语的语音识别+多种目标语言的翻译，专为 VRchat 虚拟现实社交平台设计。


### 🚀 项目特色

- ⚡ 超低延迟 - C++ 原生实现，响应速度比 Python 版本快 5-10 倍
- 💾 轻量高效 - 内存占用仅 15~17 MB，CPU占用率 0.1% ~ 0.3% ，启动时间 < 0.3秒
- 🎯 精准识别 - 集成优化的语音识别引擎，目前可识别中文普通话和英文
- 🌍 多语言翻译 - 使用 DeepSeek API 实时翻译

### 🎙️ 使用方法

1. 进入本仓库的 release 分支
2. 下载zip压缩包并解压，打开以软件名称命名的文件夹（程序本体）。
3. 找到VRChatEasyTrans-AI.exe，双击运行即可启动程序。请将此文件夹保存在安全的位置(如D盘的根目录)，避免删除和转发。建议创建快捷方式放在桌面或开始菜单。
4. 填写你的配置，以及讯飞星火和 Deepseek 的 API 认证信息（下面会讲怎么获取）
5. 点击启动，这时配置会保存，下次启动无需再次配置
6. 启动 VRchat，设置搜索 “OSC”，点击启用 OSC 功能
7. 尝试讲话，观察日志或游戏中chatbox的输出情况，根据环境噪音情况和个人讲话习惯调整静音阈值和断句时长。
8. 此软件不定期更新，想第一时间获得更新消息可以进入我的主页点击 Follow

- 讯飞星火配置（用于语音识别）
1. 注册讯飞开放平台：https://www.xfyun.cn
2. 登录后点击右上角的“控制台”
3. 创建一个应用，进入应用
4. 左边栏点击“语音识别”模块下的“语音听写”
5. 右边 Websocket服务接口认证信息，可以看到你创建的应用的APPID，APISecret 和 APIKey，复制到 VRChatEasyTrans 中对应的讯飞星火部分。
6. 对于中文和英文，每天有500次的免费额度，也就是500句话，应该够用了吧。。不够用的自己买

- Deepseek 配置（用于翻译）
1. 注册 Deepseek开放平台：https://platform.deepseek.com
2. 登陆后点击左边栏的 API keys
3. 点击 创建API key。注意，创建后API key只会立刻显示一次，请立即复制保存！将 API key 复制到 VRChatEasyTrans 中对应的Deepseek的API密钥部分。
4. 点击左边栏的 用量信息 ，查看账户余额
5. 没有余额的话点击左边栏 充值 。第一次充10块钱就行，能用好久。

### 🛠 技术架构

流式结构

```
|
├── 语音捕获、分割
├── 音频数据编码、提交语音识别模型API
├── 得到的中文文本，提交翻译模型API
├── 得到的外文文本，打包成OSC发送到VRChat
├── 文本翻译显示
V
```

技术栈

- 前端界面: Qt6 Widgets + QML
- 语音识别: 讯飞星火
- 文本翻译: Deepseek
- 音频采集: Qt Multimedia
- 网络通信: Qt Network (HTTP/OSC)
- 配置管理: 静态管理类 + 配置文件
- 构建系统: CMake
- 编译器: mingw_32

### 📦 构建与使用

系统要求

- 操作系统: Windows 10/11, macOS 10.14+, Ubuntu 18.04+
- 编译器: C++17 兼容编译器
- 依赖: Qt6.0+, CMake 3.16+

快速开始

1. 克隆项目
   ```bash
   git clone https://github.com/Rosa-11/VRCEasyTrans.git
   ```
2. 构建项目
   ```bash
   mkdir build && cd build
   cmake ..
   cmake --build . --config Release
   ```

VRchat 设置

1. 在 VRchat 中启用 OSC：
   · 设置 → 搜索 OSC → 启用 OSC
   · 默认端口：9000
2. 在 VRCEasyTrans 中的配置：
   · OSC 主机：127.0.0.1
   · OSC 端口：9000
